

# 构建工具

## 介绍

Webpack是一个开源的前端打包工具。Webpack提供了前端开发缺乏的模块化开发方式，将各种静态资源视为模块，并且它生成优化过的代码。

Webpack可以从终端、或是更改webpack.config.js来设置各种功能

要使用Webpack前需先安装Node.js。Webpack其中一个特性是使用loader来将资源转化成模块。开发者可以自定义loader的顺序、格式来因应项目的需求

## 作用

- 转化es6以上的语法
- 转化jsx
- css前缀补充/预处理器
- 压缩混淆
- 图片压缩

## 优势

- 社区生态丰富
- 配置灵活和插件化扩展
- 官方更新迭代快
