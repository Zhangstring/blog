# 学习 vue-cli

主要通过`vue inspect --mode > output.js`来查看使用到的 webpack 的配置,版本 vue-cli@4.2.0，选了默认 default 配置,由于添加了 postcss-px2rem，生成的配置中带有 postcss-px2rem，但不影响学习。先抽出相同配置，然后在看开发，生产。

## 通用
